## Overview

Scroll is the primary interaction on the web. This collection showcases various techniques for making that experience more engaging and memorable.

## Animation Patterns

- **Parallax Layers**: Multi-speed depth effects
- **Text Reveals**: Character, word, and line animations
- **Scale Transitions**: Zoom in/out on scroll
- **Pin & Progress**: Sticky sections with scroll progress
- **Horizontal Scroll**: Convert vertical to horizontal movement

## Implementation

Each pattern is documented with code examples and configuration options. Built with GSAP ScrollTrigger for reliable cross-browser performance.
