## Overview

Speed up your UX workflow with this minimal wireframe kit. Focus on structure and user flows without the distraction of visual design decisions.

## What's Included

- **Layout Templates**: Common page structures for web and mobile
- **UI Patterns**: Forms, navigation, cards, modals, and more
- **Flow Connectors**: Easy-to-use arrows and connection elements
- **Annotation Tools**: Comments and specification callouts

## Philosophy

Good wireframes communicate clearly. This kit strips away unnecessary complexity to help you focus on what matters: the user experience.
